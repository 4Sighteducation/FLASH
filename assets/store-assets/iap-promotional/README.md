## IAP Promotional Images (App Store Connect)

These are **marketing artworks** for App Store Connect **In‑App Purchase Promotional Images**.

They are intentionally **not** screenshots of the in-app paywall UI (Apple may reject UI screenshots).

### Files (SVG masters)

- `flash_premium_monthly.svg`
- `flash_premium_annual.svg`
- `flash_pro_monthly.svg`
- `flash_pro_annual.svg`

### SKU mapping (your current products)

- `flash_premium_monthly` → `flash_premium_monthly.svg`
- `flash_premium_annual` → `flash_premium_annual.svg`
- `flash_pro_monthly` → `flash_pro_monthly.svg`
- `flash_pro_annual` → `flash_pro_annual.svg`

### Export

App Store Connect typically expects a **square promotional image**. Export each SVG to:

- **PNG**
- **1024 × 1024**
- **sRGB**

If you use Figma:

- Import SVG → Select frame → Export → PNG @1x → ensure 1024×1024.

### Notes

- Avoid embedding a **price** in these images (prices vary by storefront/currency).
- Keep text **large** (these tiles are often displayed small).
